main.py：
deta.xlsx の技能実習生データを読み込み，実験用データセットを作成する前処理プログラム，全て欠損の列および「Unnamed」で始まる列を削除し，原データのバックアップを保存，「状況のまとめ」列をもとに二値ラベル「失踪の有無」を作成し，「失踪」を含むデータを1，それ以外を0として設定する。また，「入国待ち」と「その他」の状態を持つデータを除外し，実際に在留していた実習生のみを分析対象とする，その後，本研究で使用する属性を特徴量として抽出する，数値変数「入国時年齢」は中央値で補完し，カテゴリ変数の欠損値は「N/A」で補完，最終的に，機械学習実験で使用するデータセット clean.csv を生成する．

使用したデータ：
deta.xlsx

対応箇所：
卒業論文 第4章 データ前処理

clean2.py：
clean.csv を用いて，特徴量が完全に一致する実習生データに対して，重複および矛盾の有無を確認し，後続の分析用データを整理するためのプログラム．
入国時年齢，性別，職種関係，職種，実習先所在地（都道府県・市区町村），派遣会社，学校所属，組合，所属機関をグループ化条件とし，各グループ内で「失踪の有無」の分布を集計する．
次の3種類のグループを抽出する．
1．同一の属性組合せにおいて，失踪者と未失踪者が混在する矛盾グループ
2．同一の属性組合せにおいて，全員が失踪者である重複グループ
3．同一の属性組合せにおいて，全員が未失踪者である重複グループ
出力ファイルは以下である．
・jyuhukumix.csv：失踪者と未失踪者が混在する矛盾グループ
・jyuhukulost.csv：全員が失踪者である重複グループ
・jyuhukusafe.csv：全員が未失踪者である重複グループ
・jyuhuku.csv：矛盾グループに属する詳細データ
また，clean.csv に含まれる総人数，失踪人数，未失踪人数を表示し，その後，矛盾グループを除外するためのデータ作成処理を行う，clean2.csvを作成しました

使用したデータ：
clean.csv

対応箇所：
卒業論文 第4章 データ前処理

LOOCV_ALL_2.py：
clean2.csv を用いて，機械学習モデルによる失踪予測実験を行うプログラム．
決定木，ランダムフォレスト，ロジスティック回帰，SVMの4種類の分類モデルを用い，LOOCVによってモデル性能を評価する．
各サンプルを1件ずつテストデータとして取り出し，残りのデータを学習データとしてモデルを学習する LOOCV を実行する．
評価指標として，Accuracy，Precision，Recall，混同行列，Sensitivity，Specificity，および AUC を算出する．
各モデルにおいて，実際には失踪しているが非失踪と予測されたデータを抽出し，LOOCV_FN_モデル名_2.csv として保存する．

使用したデータ：
clean2.csv

対応箇所：
卒業論文 第5章 実験結果（LOOCV によるモデル評価）

LOOCV_FN2.py：
4つのモデルすべてで共通して失踪者を検出できなかった実習生を抽出し，その属性分布を分析するためのプログラム．
全モデルで共通して偽陰性となった失踪者をCommon_FN_all_models_2.csv として保存する．
さらに，共通FNに含まれる各属性について件数分布を集計し，CommonFN_counts_属性名.csv として出力する．
この結果は，現在用いている属性情報だけでは十分に識別できない失踪者の特徴を確認するために用いた．

使用したデータ：
LOOCV_FN_DecisionTree_2.csv，
LOOCV_FN_RandomForest_2.csv，
LOOCV_FN_LogisticRegression_2.csv，
LOOCV_FN_SVM_2.csv

対応箇所：
卒業論文 第6章 実験の考察

maketree.py：
clean2.csv を用いて決定木モデルを学習し，その分岐構造を可視化するためのプログラム．
DecisionTreeClassifier を用いてモデルを学習し，木の深さを max_depth=4 に制限することで，決定木を作成する．
学習した決定木モデルを plot_tree により可視化し，各分岐条件と分類結果を図として表示する,決定木の構造図を decision_tree.png として保存する．

使用したデータ：
clean2.csv

対応箇所：
卒業論文 第5章 実験結果

重要度.py：
clean2.csv を用いて，各機械学習モデルにおける失踪関連属性の重要度を抽出するためのプログラム．
決定木，ランダムフォレスト，ロジスティック回帰，SVMの4種類のモデルを学習しその後，各モデルについて以下の方法で重要属性を抽出する．
決定木：feature_importances_ による重要度，ランダムフォレスト：feature_importances_ による重要度，ロジスティック回帰：係数（coef_）の絶対値，SVM：線形カーネルの場合は係数（coef_）の絶対値，非線形カーネルの場合は permutation importance
抽出した重要度は絶対値の大きい順に並べ，上位30件をモデルごとに csv として保存する．

使用したデータ：
clean2.csv

対応箇所：
卒業論文 第6章 実験の考察
